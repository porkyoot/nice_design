# NiceGUI Hello World

A simple, premium-styled Hello World project using NiceGUI.

## Setup & Running

The project includes an automated run script that handles virtual environment creation and dependency installation.

Simply run:
```bash
./run.sh
```

The script will:
1. Create a virtual environment (`.venv`) if it doesn't exist.
2. Activate the environment.
3. Install/Update dependencies from `requirements.txt`.
4. Launch the application.

The app will be available at [http://localhost:8080](http://localhost:8080).
